# Member 2: The UI/UX Visionary

**Your Workspace:** The `/frontend` folder.

Your main job is to build the user interface using React and TailwindCSS. You do not need to wait for the backend to be finished—you will build the UI using **Mock Data** (hardcoded JSON arrays).

## Your Tasks
1. **The Navigation Shell:** Create a common layout with a Left Sidebar (Products, Operations, Dashboard, Settings, Profile Menu) and a Top Header.
2. **The Dashboard:** Go to `frontend/dashboard/page.tsx`. Use an AI tool to generate the main Dashboard view with the 4 KPIs and Recent Movements table.
3. **The Forms:** Go to `frontend/operations` and build the forms for "Receipts" (incoming goods) and "Deliveries" (outgoing goods).

## Your AI Prompts
* **Prompt 1 (Dashboard UI):** "I need a beautiful, enterprise-grade Next.js Dashboard using TailwindCSS and Shadcn UI. It needs a left sidebar and a main section with 4 KPI metric cards ('Total Products', 'Low Stock', 'Pending Receipts', 'Pending Deliveries'). Below that, create a responsive table showing 'Recent Inventory Movements'."
* **Prompt 2 (Forms):** "Create a Next.js form component for an 'Incoming Stock Receipt'. It needs fields for Supplier Name, a dropdown to select a Product (with fake data for now), a Quantity input, and a Submit button. Style it to look modern and accessible using Tailwind."
